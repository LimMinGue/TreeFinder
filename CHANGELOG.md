# 변경 이력 (Changelog)

## v0.3.2 — 2026-07-23

### ✨ 새 기능
- **트리 경로 복사** — 왼쪽 트리의 폴더·즐겨찾기 우클릭 메뉴에 "경로 복사"가 추가되었습니다. 파일 목록의 경로 복사와 동일하게 한글 경로 보정(NFC)이 적용됩니다

### 🐛 수정
- **드롭스택에서 끌어다 놓으면 묻지 않고 이동되던 문제** — 파일 목록·아이콘 보기·탭·트리 어디에 놓든 "여기로 이동 / 여기로 복사 / 취소" 메뉴로 직접 선택합니다(ESC·바깥 클릭 = 취소). "여기로 이동"은 다른 디스크로도 이동됩니다
- **드롭스택이 작업 후에도 남아 다시 옮기면 "존재하지 않음" 오류가 나던 문제** — 이동·복사가 전부 완료되면 스택이 자동으로 비워집니다. 취소하거나 실패하면 모아둔 항목은 유지되고, 드래그를 시작할 때 이미 사라진 파일은 자동 정리됩니다(응답 없는 네트워크 장비 항목은 정리 판정을 생략해 앱이 멈추지 않습니다)
- **터미널 탭에 파일을 놓으면 조용히 사라지던 동작** — 터미널 탭은 드롭을 받지 않고 원위치로 돌려보냅니다

## v0.3.1 — 2026-07-23

### ✨ 개선
- **터미널 최초 열기 = 보고 있는 폴더에서 시작** — 터미널을 처음 열면(마지막 탭을 닫고 다시 열 때 포함) 현재 보고 있는 폴더에서 셸이 바로 시작됩니다. "현재 폴더로 이동"을 다시 누를 필요가 없습니다. + 새 탭은 종전대로 홈에서 시작. 네트워크 볼륨·진입 권한 없는 폴더·사라진 폴더는 안전을 위해 홈에서 시작(응답 없는 네트워크 장비가 앱을 멈추지 않도록)
- **터미널 Restart Shell = 제자리 재시작** — 현재 폴더에서 연 터미널을 재시작해도 같은 폴더에서 다시 시작합니다(이전에는 홈으로 이동)

### 🐛 수정
- **듀얼 페인에서 빈 공간 클릭 시 포커스(활성 페인)가 전환되지 않던 문제** — 파일 목록의 빈 공간을 클릭해도 그 페인이 활성화되어 왼쪽 트리 동기화·터미널 "현재 폴더로 이동"이 해당 페인 기준으로 동작합니다. 이미 선택된 행을 다시 클릭할 때도 동일하게 전환

## v0.3.0 — 2026-07-23

### ✨ 새 기능
- **Finder식 검색(Spotlight)** — 도구모음 검색이 하위 폴더 전체를 **파일 이름 + 내용**으로 즉시 검색합니다(Spotlight 색인 조회 — 기존 자체 순회는 대형 폴더에서 수십 초 + 이름만 매칭). 결과에는 파일이 **들어있는 폴더**가 회색으로 함께 표시되어 같은 이름 파일을 구분할 수 있고, 폭이 좁으면 …으로 줄여 표시합니다. 색인이 없는 볼륨에서는 자동으로 파일명 순회 검색으로 대체
- **색상 태그(Finder 태그)** — 우클릭 메뉴 "태그"에 색 점 7개(빨·주·노·초·파·보·회)와 지우기(×)가 한 줄로 표시됩니다. 클릭 한 번으로 선택 항목 전체에 일괄 지정/해제, 현재 색에는 파란 링 표시, ⌘Z 되돌리기 지원. Finder 태그와 완전 호환되며 목록 행 색과 왼쪽 트리 폴더 색에 즉시 반영
- **Finder 키보드 규약** — 선택 항목에서 **⌘↓ = 열기/폴더 진입**(리스트·아이콘 뷰), **Enter = 이름 변경**(리스트). 열기는 더블클릭·⌘O도 종전대로
- **새 폴더 = 이름 변경 상태로 생성** — "새 폴더"를 만들면 바로 이름을 입력할 수 있는 편집 상태로 생성됩니다(Finder 규약)
- **트리 네트워크에 발견 컴퓨터 표시** — 왼쪽 트리의 "네트워크"를 펼치면 목록과 동일하게 실제 발견된 컴퓨터들이 표시되고, 클릭하면 바로 연결합니다. 기억된 공유는 그 아래 유지

### 🐛 수정
- **왼쪽 트리가 자동 새로고침되지 않던 문제** — 폴더 생성·삭제·이름변경이 트리에 즉시 반영됩니다. 펼쳐둔 하위 폴더는 그대로 유지
- **"개인/가족"처럼 슬래시(/)가 든 이름을 지을 수 없던 문제** — macOS 규약(표시 `/` ↔ 디스크 `:`)대로 저장하고, 목록 표시도 트리·Finder와 같이 `/`로 통일
- **폴더를 보는 중 하위에서 파일이 계속 바뀌면 화면이 주기적으로 깜빡이던 문제** — 새로고침을 "바뀐 행만 제자리 교체"로 바꿔 목록·미리보기·트리가 잔잔해졌습니다(빌드·서버 폴더에서 체감). 선택 중인 파일 내용이 바뀌어도 미리보기는 다시 로드하지 않습니다(Finder 동일 — 재선택 시 갱신)
- **네트워크 연결 창을 취소해도 한 번 더 뜨던 문제** — 더블클릭·연결 진행 중 재클릭으로 연결이 중복 시작되던 것을 차단
- **네트워크에 같은 공유(home)가 2개로 보이던 문제** — 같은 공유의 다른 주소 표기를 하나로 병합(기존 저장분도 자동 정리)
- **네트워크 연결 실패 후 목록 파일을 선택해도 트리가 네트워크에 남아 있던 문제** — 목록에서 항목을 선택하면 트리가 그 위치로 따라옵니다
- **폴더의 커스텀 아이콘을 지워도 트리에 반영되지 않던 문제** — 커스텀 아이콘·태그 변경이 실행 중에도 트리에 반영됩니다

## v0.2.0 — 2026-07-21

### ✨ 새 기능
- **터미널 테마 선택** — 설정 ▸ 터미널에 테마 항목 추가. 기본 · Solarized 다크 · Solarized 라이트 · Dracula · Nord 5종(16 ANSI 색 + 전경/배경/커서). 선택 즉시 열려 있는 모든 터미널 세션에 반영되며, 세션 둘레 여백 색도 함께 맞춰집니다. 기본값은 기존과 같은 검정 배경이라 업데이트로 모양이 바뀌지 않습니다
- **`.sh` 더블클릭 = 터미널에서 실행** — 파일 목록에서 셸 스크립트를 더블클릭하면 우측 패널에 새 터미널 탭이 열리고(패널이 접혀 있으면 자동으로 펼침) 스크립트가 있는 폴더로 이동해 실행합니다. 실행 권한이 없으면 `sh 경로`로 실행. 탭 이름은 스크립트 파일명. ⌘O도 같은 경로
- **Broadcast input** — "현재 폴더로 이동" 버튼 옆 토글. 켜면 활성 터미널에 입력한 키가 열려 있는 **모든 터미널 탭**에 동시에 전달됩니다(서버 여러 대에 같은 명령 실행). ⌘단축키·방향키는 제외
- **명령 도움말 패널 껐다 켜기** — 터미널 하단 도움말 밴드를 접는 토글 버튼 추가. 끈 상태에서는 명령을 실행해도 다시 열리지 않습니다

### ✨ 개선
- **터미널 시작 위치 = 홈 폴더** — 새 터미널 탭은 항상 `~`에서 시작합니다. 이전에는 앱 프로세스의 작업 폴더(Finder로 실행하면 `/`)를 물려받아 위치가 일정하지 않았습니다. "현재 폴더로 이동"과 `.sh` 실행은 그대로 해당 폴더로 이동
- **터미널 탭 제목** — 로컬은 **현재 경로**(`~/Dev/TreeFinder`), `ssh`/`mosh` 접속 중에는 **접속 대상**(`user@host`)을 표시. 원격 세션에서 빠져나오면 OSC7 호스트로 감지해 자동 복귀. 더블클릭 수동 이름은 그대로 최우선
- **파일 선택 시 미리보기 탭으로 자동 전환** — 터미널 탭을 보고 있어도 파일 목록에서 파일을 선택하면 미리보기로 전환됩니다(선택 해제는 전환하지 않아 터미널 작업을 가로채지 않음)

### 🐛 수정
- **`exit`으로 셸을 끝내도 터미널 탭이 남던 문제** — 셸이 종료되면(`exit`·`logout`·비정상 종료) 해당 탭도 함께 닫힙니다. 이미 끝난 세션이라 확인 시트는 뜨지 않습니다
- **터미널 탭이 1개일 때 닫을 수 없던 문제** — 마지막 탭에도 ✕가 표시되고 닫을 수 있습니다. 전부 닫으면 미리보기 탭으로 돌아가고, 다시 터미널 탭을 열면 새 셸이 시작됩니다. (파일 목록 탭은 종전대로 마지막 1개를 남깁니다 — 그쪽은 ⌘W로 창을 닫는 규약)

### 🛠️ 개발
- **`build.sh` 추가** — README의 `xcodebuild` + `ditto` + `open` 절차를 스크립트 하나로. `./build.sh`(Release) · `-d`(Debug) · `-i`(설치) · `-r`(실행) · `-h`(도움말). `DEVELOPER_DIR` 자동 지정, Metal 툴체인 누락 시 안내 출력, 설치 전 실행 중인 앱 종료. `TF_SIGN_IDENTITY="인증서 이름"`을 주면 설치 전 재서명해 **전체 디스크 접근 승인이 재빌드에도 유지**됩니다(README 각주 자동화)

## v0.1.2 — 2026-07-18

### ✨ 새 기능
- **압축파일 내부 목록 미리보기** — zip·tar·tgz·bz2·xz·jar 등 압축파일을 선택하면 미리보기 패널에 **내부에 담긴 파일 목록**을 표시(표시 전용 — 추출·실행 없음). 항목 수 헤더 + 형제 파일 그룹화 정렬, ⌘F로 항목 찾기. macOS zip의 `__MACOSX`·`.DS_Store` 잡음 자동 제외, 대용량·손상·암호 아카이브와 비UTF-8 파일명도 안전 처리(압축폭탄 메모리 방어·타임아웃·손실 허용 디코딩), 디스크 이미지(dmg·iso)·설치 패키지(pkg)는 제외

### 🐛 수정
- **터미널 도움말 밴드가 화면을 덮던 문제** — 터미널 탭을 추가한 뒤 창 크기를 조절하면 하단 명령 도움말 영역이 화면 전체를 덮던 현상 수정(추가 탭이 있을 때 발생). 세션 교체가 분할 레이아웃을 깨지 않도록 구조 개선 + 도움말 밴드 크기 고정
- **터미널 글자 잘림** — 터미널 내용이 가장자리에 너무 붙어 잘려 보이던 문제를 내부 여백(패딩) 추가로 개선
- **미리보기 확대 시 왼쪽 잘림** — 확대하면 화면 중앙 기준으로 커져 왼쪽이 잘리고 옆으로 스크롤해야 하던 것을, **왼쪽 끝 정렬**로 변경(세로 위치는 유지). 텍스트·이미지·문서 미리보기 공통

- **하단 경로 바** — 로컬라이즈드 브레드크럼, 세그먼트 클릭 즉시 이동 · ⌘L 주소 입력 · ⇧⌘G 폴더로 이동 · ⌘K 서버에 연결
- **네트워크 브라우즈** — 사이드바 네트워크 선택 시 주변 SMB 컴퓨터 발견(Bonjour), 더블클릭 = 연결(인증·공유 선택은 시스템 UI). 마운트한 공유 자동 기억·끊김 흐림 표시·원클릭 재연결·인라인 ⏏
- **사이드바** — Finder식 모노크롬 심볼 아이콘, 즐겨찾기(핀 해제), 휴지통·볼륨 직접 접근, USB/네트워크 마운트 자동 반영, 태그 색 폴더는 지정 색으로 표시
- **뒤로/앞으로** (⌘[ / ⌘]) · 상위 폴더(⌘↑) · 숨김 파일 토글(⇧⌘. — 목록과 트리 동기)

### 👀 보기
- **리스트 · 아이콘 · 갤러리 뷰** (⌥⌘1/2/3) — 갤러리 = 대형 미리보기 + 필름스트립, 탭별 뷰 상태
- **전 컬럼 정렬** — 이름·수정일·생성일·크기·종류, 폴더 우선 불변 · **헤더 구분선 더블클릭 = 컬럼 적정 폭**(Excel식)
- **폴더 크기 표시** — 백그라운드 재귀 계산(단일 actor), 크기 정렬에 폴더 참여, 부분 측정 "≥" 정직 표기, 선택 합산 상태바
- **태그 풀 로우 컬러** — Finder 라벨 색을 행 전체 배경으로
- **격자(교차 행) 배경** · 잘라내기 행 흐림 · 오늘/어제 상대 날짜

### 👁 미리보기 · 편집
- **미리보기 패널** — QuickLook + 파일 정보 테이블(이미지는 EXIF: 해상도·카메라·조리개·노출·ISO), 핀치·버튼 확대/축소(0.25×~5×)
- **내용 기반 텍스트 미리보기** — 확장자가 아니라 내용(UTF-8)으로 판정: `.gitignore`·`LICENSE`·`.yml` 등도 텍스트로 표시
- **HWP/HWPX 미리보기** — 전 페이지 SVG 렌더(rhwp 동반), 실패 시 내장 미리보기 리소스 폴백
- **마크다운 WYSIWYG 편집** — .md 선택 즉시 렌더+편집, ⌘S 저장, 미저장 이동 시 확인(Toast UI Editor 오프라인 동반)

### ⌨️ 내장 터미널 (SwiftTerm)
- **멀티 탭** — 독립 셸 세션, 전환해도 세션 유지, **✕ 닫기 확인**(실행 중 작업 보호)
- **자동 탭 제목** (iTerm2식) — 실행 중 명령·현재 경로 표시, **더블클릭 = 이름 직접 지정**(세션 한정)
- **파일 드롭 = 전체 경로 입력** — 인용 이스케이프, 실행 없음(Terminal.app 규약)
- **명령 도움말 밴드** — 하단 상시 표시, vi·grep·tar 등 기본 명령 ~30종 실행 시 한국어 치트시트로 자동 교체(크기 조절)
- **"현재 폴더로 이동" = 항상 새 탭** — vi 등 실행 중 프로그램에 cd가 주입되는 오염 원천 차단
- **터미널 폰트 설정** — 고정폭 패밀리·크기(Nerd Font 파워라인 글리프)

### 🧰 파일 작업
- 복사/이동/휴지통/복제/압축(zip)/인라인 이름 변경 — **Undo**, 진행률 패널+취소, 덮어쓰기 없는 충돌 회피 명명
- **되돌려 놓기** — TreeFinder가 지운 파일을 원위치로 복원(공개 API만 사용)
- **정보 가져오기** (⌘I) — Finder식 정보 창: 폴더 실측 크기·위치·EXIF·권한 표(읽기 전용), 항목당 창(최대 10)
- **Drop Stack** — 사이드바 임시 선반에 모아뒀다 일괄 이동/복사
- **클립보드 NFC 보정** — 경로 복사 시 한글 자모 분리 방지
- 빈 공간 우클릭 메뉴 — 새 폴더·새 문서·붙여넣기·새 탭·터미널·정렬·보기 옵션·새로 고침·정보

### ⚠️ 알려진 한계
- ad-hoc 서명 로컬 빌드 전용(공증·배포 없음) — 직접 빌드해서 사용
- 터미널 명령 감지는 키 입력 기반 — 히스토리(↑)·붙여넣기로 실행한 명령은 도움말 밴드 미반영
- 텍스트 미리보기는 UTF-8 한정(EUC-KR 등 레거시 인코딩은 QuickLook 폴백), 1MB 초과분 생략 표기
- 네트워크 볼륨의 폴더 크기 계산은 v1 제외(행업 방지) · 컬럼 뷰 미구현 · 트리 펼침 상태는 복원 대상 아님
